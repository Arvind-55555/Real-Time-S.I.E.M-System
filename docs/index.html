Detection Capabilities

Authentication Attacks

Brute Force Attack: Detects >10 failed login attempts within 5 minutes
Credential Stuffing: Identifies attempts from 20+ unique IPs

Data Exfiltration

Large Data Transfer: Flags transfers >100MB
DNS Tunneling: Detects abnormally long DNS queries

Network Attacks

Port Scanning: Identifies >20 port access attempts
DDoS Detection: Triggers on >1000 requests/second

Web Attacks

SQL Injection: Regex-based detection of injection patterns
XSS Attempts: Identifies script injection attempts

ML-Based Detection

Anomaly Detection: ML model detects statistical anomalies with 85-95% accuracy

Quick Start

1. Clone Repository

git clone https://github.com/Arvind-55555/Real-Time-S.I.E.M-System.git cd Real-Time-S.I.E.M-System

2. Automated Setup

bash setup_production.sh

3. Start Collecting Data

# Terminal 1: Start web data collector python3 scripts/collect_web_data.py --mode continuous # Terminal 2: Start dashboard python3 scripts/run_live_dashboard.py # Browser: Open http://localhost:8080
